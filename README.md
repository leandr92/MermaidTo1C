# Mermaid.js транспилированный для 1С

Этот проект транспилирует Mermaid.js для использования в старом WebKit платформы 1С.

## Установка

```bash
npm install
```

## Сборка

```bash
npm run build
```

Собранный файл будет находиться в `dist/mermaid_all_in_one.js`.

## Использование

Подключите собранный файл в HTML:

```html
<script src="dist/mermaid_all_in_one.js"></script>
```

Mermaid автоматически инициализируется и обработает все элементы с классом `.mermaid`.

## Особенности

- Транспилирован в ES5 для совместимости с IE 11 и старыми браузерами
- Все зависимости упакованы в один файл
- Автоматическая инициализация при загрузке
- Полифиллы для отсутствующих API включены автоматически

## Структура проекта

- `src/index.js` - точка входа для сборки
- `rollup.config.js` - конфигурация Rollup
- `.babelrc` - конфигурация Babel для транспиляции в ES5
- `dist/mermaid_all_in_one.js` - собранный файл (создается после сборки)
